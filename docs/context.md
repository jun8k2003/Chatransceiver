# プロジェクト・コンテキスト (docs/context.md)

このファイルは、アプリケーションの背景、目的、ターゲットユーザー、コアバリュー、および全体的な技術スタックを管理します。

## 1. プロジェクト概要
* **アプリケーション名**: Chatransceiver (暫定)
* **概要**: チャットUIの視認性と、トランシーバーの即時性・操作性を融合した「半二重通信グループボイスチャットアプリ」。

## 2. 目的・背景
* テキストチャットの手軽さ・ログの残りやすさと、音声通話の即時性（打鍵不要で手が離せない時でも使える）を両立させる。
* 「電話（双方向同時通話）」のように拘束されることなく、トランシーバーのように「用があるときだけプッシュして話しかける」非同期かつ即時性の高いコミュニケーションを実現する。

## 3. ターゲットユーザー / ユースケース
* 現場作業者（工場、店舗、イベント運営など、手が離せない状況で複数人に一斉連絡したいユーザー）
* ゲームでのチーム連携や、家族・友人同士のラフな常時接続・声の掛け合い

## 4. コア機能・提供価値
* **プッシュ・トゥ・トーク (PTT) 操作**: 画面上のボタンを押している間だけ音声送信。
* **チャットUI**: 過去の発話履歴がタイムライン形式で残る。
* **半二重通信**: 誰かが話している間はチャンネルが占有され、他の人は聞くモードになるトランシーバー仕様。
* **グループ送信**: チャンネルに所属するメンバー全員への一斉音声配信。

## 5. 技術スタック (暫定)
* **フロントエンド**: HTML / Vanilla CSS / **TypeScript (Vite ビルド環境)**
* **バックエンド / データベース**: **Supabase (PostgreSQL, Storage, Supabase Auth, Realtime)**

  * 自前でのサーバー構築は行わず、Supabaseの無料枠を活用したサーバーレス構成。

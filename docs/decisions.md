# 意思決定ログ (docs/decisions.md)

このファイルは、議論の末に決定した事項とその理由を記録します。

## DEC-023: グループ名（表示上のあだ名）の導入 (2026-06-11)
* **決定**: `chat_rooms` テーブルに `name text` 列（null許容）を追加し、グループに任意の表示名を付けられるようにする。
* **位置づけ**: グループの同一性はあくまで「メンバー構成」で決まる（DEC-011は維持）。`name` は表示上のあだ名にすぎない。
* **編集権限**: メンバーなら誰でも編集可能（グループ削除が全員可能である既存の思想と一貫させる）。RLSは `authenticated` ロールにUPDATEを許可。
* **フォールバック**: `name` が NULL の場合は従来どおりメンバー名のカンマ羅列を表示する。これにより既存グループ・旧フロントエンドとの後方互換を確保。
* **編集UI**: チャットヘッダーのグループ名横の ✏️ ボタンからインライン編集（Enter/フォーカスアウトで確定、Escでキャンセル）。

## DEC-024: UIリデザイン方針 (2026-06-11)
* **決定**: `mockup/proposal.html` のデザインを本番適用する。主な変更点は以下。
  * **リスト統合**: 左ペイン（個別）と中央ペイン（グループ）を1つのリストに統合し、フィルタチップ（すべて/個別/グループ）で絞り込む。上下スプリッターは廃止。
  * **選択モデル変更**: 行のタップ/クリックで即チャットを開く。複数人選択での新規グループ作成は「＋新規グループ」ボタンから入る専用の選択モードに分離。
  * **ヘッダー簡素化**: コミュニティ接続コンボボックスの常時表示をやめ、コミュニティ名ピルのクリックで開くドロップダウン（接続履歴・ID入力・招待リンクコピー・切断・退会）に集約。
  * **PTTボタン**: スマホでは下段中央の大型円形ボタン（主役）。PCでは控えめなサイズでフッター左端（サイドバーからのマウス動線を短縮）。
  * **環境設定の整理**: コミュニティ関連操作をドロップダウンへ分離し、設定モーダルは個人設定のみ（セクション分け、トグルスイッチ化、保存ボタンは下部固定フッター）。
  * **配色**: アクセントを「青（操作系）＋オレンジ（PTT・録音系）」の2色に整理。半透明＋blurをやめフラットな塗りに変更。
  * **削除ボタンの表示**: PCはホバー時のみ表示、タッチ端末（`hover: none` または `pointer: coarse`）は常時表示。
* **不変条件**: アプリの代表ロゴ（public/chatora.png 茶トラ猫）は変更しない。
